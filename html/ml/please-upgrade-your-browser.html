നിങ്ങൾ ഇന്റർനെറ്റ് എക്സ്പ്ലോററിന്റെ പഴയ പതിപ്പാണ് ഉപയോഗിക്കുന്നത് (IE11 അല്ലെങ്കിൽ അതിൽ താഴെ അല്ലെങ്കിൽ ഈ കേർണൽ ഉപയോഗിക്കുന്ന ഒരു ബ്ര browser സർ). നിങ്ങളുടെ ബ്ര .സർ അപ്‌ഗ്രേഡുചെയ്യുന്നതുവരെ നിങ്ങൾക്ക് ഈ വെബ്‌സൈറ്റ് ആക്‌സസ് ചെയ്യാൻ കഴിയില്ലെന്നാണ് ഇതിനർത്ഥം.

ദയവായി ശ്രദ്ധിക്കുക: വിൻഡോസ് എക്സ്പി, ഇൻറർനെറ്റ് എക്സ്പ്ലോററിന്റെ പഴയ പതിപ്പുകൾ എന്നിവയ്ക്കുള്ള പിന്തുണാ സേവനങ്ങൾ നിർത്തലാക്കി

ജനുവരി 12, 2016 വരെ, മൈക്രോസോഫ്റ്റ് ഐ‌ഇ 11 നും അതിനു താഴെയുമുള്ള അനുബന്ധ പിന്തുണയും അപ്‌ഡേറ്റുകളും മേലിൽ നൽകില്ല. നിർണായക ബ്ര browser സർ സുരക്ഷാ അപ്‌ഡേറ്റുകൾ ഇല്ലാതെ, നിങ്ങളുടെ ബിസിനസ്സ് ഡാറ്റയും വിവരങ്ങളും മോഷ്‌ടിക്കാനോ കേടുവരുത്താനോ സാധ്യതയുള്ള ദോഷകരമായ വൈറസുകൾ, സ്‌പൈവെയർ, മറ്റ് ക്ഷുദ്ര സോഫ്റ്റ്‌വെയർ എന്നിവയ്‌ക്ക് നിങ്ങളുടെ കമ്പ്യൂട്ടർ ഇരയാകാം. കാണുകഇന്റർനെറ്റ് എക്സ്പ്ലോററിന്റെ പഴയ പതിപ്പുകൾക്കുള്ള മൈക്രോസോഫ്റ്റിന്റെ പിന്തുണ അവസാനിച്ചുവെന്നത് ശ്രദ്ധിക്കുക.

കൂടുതൽ വിപുലമായ ബ്ര browser സർ

ഇനിപ്പറയുന്ന ബ്രൗസറുകളുടെ ഏറ്റവും പുതിയ പതിപ്പ് ഉപയോഗിക്കാൻ ശുപാർശ ചെയ്യുന്നു. നിങ്ങളുടെ കമ്പ്യൂട്ടറിൽ ഇനിപ്പറയുന്ന ബ്ര browser സറിന്റെ ഏറ്റവും പുതിയ പതിപ്പ് ഉണ്ടെങ്കിൽ, സ്പീഡ് മോഡ് ആക്സസ് ചെയ്യുന്നതിന് നിങ്ങൾക്ക് നേരിട്ട് ബ്ര browser സർ ഉപയോഗിക്കാംഅത്രയേയുള്ളൂ.

എന്തുകൊണ്ടാണ് ഈ പേജ് ദൃശ്യമാകുന്നത്?

ബ്ര browser സർ അപ്ഗ്രേഡ് ചെയ്യുന്നതിന്റെ അർത്ഥമെന്താണെന്ന് നിങ്ങൾക്കറിയില്ലെങ്കിൽ, കമ്പ്യൂട്ടർ പ്രവർത്തനത്തിൽ പ്രാവീണ്യമുള്ള ചില സുഹൃത്തുക്കളോട് ചോദിക്കുക. നിങ്ങൾ IE6 / 7/8/9/10 അല്ല, 360 ബ്ര browser സർ, ക്യുക്യു ബ്ര browser സർ, സോഗോ ബ്ര browser സർ മുതലായവ ഉപയോഗിക്കുന്നില്ലെങ്കിൽ, നിങ്ങൾ ബ്ര browser സറിന്റെ ഏറ്റവും പുതിയ പതിപ്പ് ഉപയോഗിക്കാത്തതിനാൽ ഈ പേജ് ദൃശ്യമാകുന്നു, ഏറ്റവും പുതിയ കാനിലേക്ക് നവീകരിക്കുക.

IE6, IE7, IE8, IE9, IE10 എന്നിവ ഒരുമിച്ച് ബഹിഷ്‌കരിക്കുക
